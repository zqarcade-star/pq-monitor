# 공고 첨부파일 AI 분석기 — 개발 계획

> 마지막 업데이트: 2026-04-10

---

## 한 줄 요약

나라장터 공고 첨부파일(HWP/PDF/Excel)을 웹에 업로드하면
Gemini AI가 팀별 맞춤 질문에 자동으로 답변해주는 도구

---

## 배경 & 제약

- G2B API는 첨부파일 다운로드 엔드포인트를 제공하지 않음
- 나라장터 웹 크롤링은 2025년 1월 개편 후 차단됨
- **결론:** 팀원이 직접 파일을 다운받아 업로드하는 방식이 유일한 현실적 방법

---

## 확정된 아키텍처

```
팀원이 나라장터에서 첨부파일 직접 다운로드
    ↓
Streamlit 웹 도구에 파일 업로드 (HWP / PDF / Excel, 다중 파일 가능)
    ↓
팀 선택 체크박스 (공통 / 기획 / PQ / 제안)
    ↓
Google Gemini Flash API로 분석 (무료)
    ↓
결과 즉시 표시 + 복사 버튼 (Excel/Sheets 붙여넣기용)
```

---

## 기술 스택

| 항목 | 선택 | 이유 |
|---|---|---|
| 웹 프레임워크 | Streamlit | Python 기반, 빠른 개발, 무료 호스팅 |
| 호스팅 | Streamlit Community Cloud | 무료, 외부 접근 가능, 이메일 접근 통제 |
| AI | Google Gemini Flash | 무료 (하루 1,500건), 한국어 우수 |
| HWP 처리 | hwp5txt | 텍스트 추출 (PDF 변환 불필요) |
| PDF 처리 | pdfplumber | |
| Excel 처리 | pandas / openpyxl | |

---

## 팀별 질문 설정 (Google Sheets [설정] 탭으로 관리)

| 팀 | 항목명 | AI 질문 | 유형 |
|---|---|---|---|
| 공통 | 발표여부 | 발표평가 또는 PT 평가가 있나요? | O/X |
| 공통 | 공고유형 | 단순PQ / 발표PQ / SOQ포함 중 선택 | 선택 |
| 제안팀 | 용역기간 | 용역 수행 기간은 몇 개월인가요? | 텍스트 |
| 제안팀 | 별도제안서여부 | 사업수행능력평가서 외 별도 제안서가 있나요? | O/X |
| 제안팀 | 제안서페이지수 | 제안서 최대 페이지 수는? | 텍스트 |
| 제안팀 | 제안서제출일 | 제안서 제출 기한은? | 텍스트 |
| 제안팀 | 면접여부 | 면접 평가가 있나요? | O/X |
| 제안팀 | 면접일자 | 면접 예정 일자는? | 텍스트 |
| 기획팀 | 추후 팀과 협의 후 추가 | | |
| PQ팀 | 추후 팀과 협의 후 추가 | | |

> 질문 추가/수정은 Google Sheets [설정] 탭에서 누구나 직접 가능

## 결과 출력 방식

- Google Sheets 자동 저장 없음
- 웹 화면에서 즉시 확인
- 복사 버튼으로 Excel / Google Sheets에 붙여넣기

---

## 접근 통제

- Streamlit Community Cloud 이메일 화이트리스트 방식
- 허용된 이메일 계정만 접근 가능

---

## 준비 상황

| 항목 | 상태 |
|---|---|
| Google AI Studio 계정 | ✅ 있음 |
| Gemini API 키 | ⏳ 미발급 → aistudio.google.com에서 발급 필요 |
| GitHub 레포 Public 전환 | ⏳ 예정 (코드 내 민감정보 없음 확인 완료) |
| 실제 공고 HWP/PDF 샘플 파일 | ⏳ POC용으로 1개 필요 |

---

## 다음 작업 순서

1. **Gemini API 키 발급** — `aistudio.google.com` → Get API key → Create API key
2. **샘플 파일 준비** — 나라장터 등록공고 첨부파일 1개 다운로드
3. **POC 테스트** — 텍스트 추출 품질 + Gemini 분석 결과 확인
4. **Streamlit 본개발** — POC 통과 후 시작
